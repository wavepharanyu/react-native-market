# Getting Started

## Step 1: Start the Metro Server

```bash
# using npm
npm start

# OR using Yarn
yarn start
```

## Step 2: Start your Application

### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```

## Login / Register Page ##

หน้า Login เพื่อทำการเข้าสู่ระบบ สำหรับจัดการสินค้า
```
username: john@gmail.com
password: password
```
หรือจะทำการลงทะเบียนที่หน้าหน้า Register ก็ได้

## Product List / Product Detail Page ##
